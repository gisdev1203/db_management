# DBER | 基于实体连接图的数据库设计工具

[English](README.md)

## 网址及演示

https://dber.tech

## 功能

1. 可视化数据库结构设计
2. 拖拽生成模型引用关系
3. 一键导出SQL语句

## 技术栈

React

Next.js

DBML

ArcoDesign

## 开始

克隆本仓库或者下载代码.

安装依赖.

```bash
npm install
# or
yarn install
```

启动开发服务:

```bash
npm run dev
# or
yarn dev
```

使用浏览器打开 [http://localhost:3000](http://localhost:3000) 查看结果.
